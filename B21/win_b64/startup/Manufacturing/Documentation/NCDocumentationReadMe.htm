IMPORTANT WARNING

Some macro files are delivered with the CATIA Manufacturing V5 products for sample purposes. They deal with: 

  • the generation of a HTML documentation related to the description of a Manufacturing program: 

Document_sample_NT.CATScript (NT version) 
Document_sample_UNIX.CATScript (UNIX version) 

  • the generation of a HTML documentation related to the description of the tools used by a Manufacturing program:

Tool_List_sample_NT.CATScript (NT version) 
Tool_List_sample_UNIX.CATScript (UNIX version) 
These macro files call CATIA Manufacturing IDL interfaces to read, search, write or modify data in a Manufacturing Process document. 
These IDL interfaces are delivered for sample purposes. 
Note that all these Manufacturing IDL interfaces are subject to change in the next releases. These changes might concern their function as well as their interface. 
No compatibility of the present set of Manufacturing IDL interfaces is guaranteed with future releases.

VBScript Usage
Set Up ManufacturingSetup
Find the Set Ups of a Manufacturing process Set ActivityRef = MfgDoc1.GetItem("Process")
If (ActivityRef.IsSubTypeOf("PhysicalActivity")) Then
    Set childs = ActivityRef.ChildrenActivities
    quantity = childs.Count
    For I=1 To quantity
       Set child = childs.Item(I)
       If (child.IsSubTypeOf("ManufacturingSetup")) Then
      Set CurrentSetup = child ......
Read the name of the Set Up Name1 = Mysetup.Name
Read the comment of a Set Up Comment1 = Mysetup.Comment
Retrieve the CATProduct product  associated to the Set Up Product1 = MySetup.Product
Retrieve machine object associated to the Set Up Machine1 = MySetup.Machine
Retrieve the list of programs of a Set Up Dim ListOfPrograms1 As MfgActivities
ListOfPrograms1 = MySetup.Programs
Machine ManufacturingMachine
Read the name of the Machine Name1 = MyMachine.Name
Read the comment of a Machine Comment1 = MyMachine.Comment
Read the information defined on the Machine (get a CKE object) Set ParmInitSpindX =
 MyMachine.GetAttribute("MFG_X_INIT_SPNDL")
InitSpind = ParmInitSpindX.Value
Read the name of the associated Tools catalog MyMachine. PreferedToolCatalogName = "Mycatalog"
Read the name of the PP words table MyMachine. PPTableName = "MyTable.PPtable"
Program ManufacturingProgram
Read the name of the program Name1 = MyProgram.Name
Read the comment of the program Comment1 = MyProgram.Comment
Read the list of activities contained in the program Set ActivityList = CurrentProgram.Activities
ManufacturingActivity ManufacturingActivity
Read the name of an activity Name1 = MyActivity.Name
Read the type of the activity ActivityType = CurrentActivity.Type
If (ActivityType = "ToolChange" ) Then …
Retrieve the tool associated to an activity Dim CurrentTool As ManufacturingTool
Set CurrentTool = CurrentActivity.Tool
Read the information defined on a Machining Operation (get the CKE object) Dim PlungeTip  As AnyObject
Set PlungeTip = Mo.GetAttribute("PlungeTip")
PlungeTipVal = PlungeTip.Value
Read the name  an activity Name1 = MyMfgOperation.Name
Read the comment  an activity Comment1 = MyMfgOperation.Comment
Tool ManufacturingTool
Read the name of a tool Name1 = MyTool.Name
Read the comment of a tool Comment1 = MyTool.Comment
Read the type of a tool TypeString = MyTool.ToolType
Read the number of a tool ToolNumber = MyTool.ToolNumber
Read the number of attributes  of a tool Dim TabAtt()
dimTab=CurrentTool.NumberOfAttributes
 Redim TabAtt(dimTab)
Read the list of names of the attributes (as CKE object names) of a tool call CurrentTool.GetListOfAttributes(TabAtt)
Read the list of names of the units of each of the attributes of a tool call CurrentTool.GetListOfAttributeUnits(TabAttUnit)
Read the information defined on a tool (get a CKE object) Set ToolDiamParam =
 MyTool.GetAttribute("MFG_DIAMETER")
ToolDiam = ToolDiamParam.Value
