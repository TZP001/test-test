Information about cutter compensation

Info about used tools

The T-Slotter used in the groove milling operations needs different cutter compensation settings. There are 2 cutting points used which are set by different length registers for T-Slotter: one for the compensation on top (No2) and one for the compensation on bottom (No1))

 

 

Reference MSG-MFG Cutter Compensation with CE (Controller Emulator)

 

Compensation Output/CUTCOM

Compensation Output None:  There is no correction to calculate for CE

 

Compensation Output 2D radial profile:

 

•CE must recalculate the toolpath from effective Contour and Radius
•Relative user radius should be deactivated
•This mode is not supported in this delivered scenario with CE

 

Compensation Output 2D radial tip:  

PPRINT MACHINE OPERATION = Profile Contouring

PPRINT OPERATION NAME    = Profile Contouring.3 (Comp. 2D Radial tip)

PPRINT TOOL ASSEMBLY     = T1 End Mill D 10

MO_INIT/Profile Contouring,Profile Contouring.3 (Comp. 2D Radial tip),T1 End Mill D 10

$$ OPERATION NAME : Profile Contouring.3 (Comp. 2D Radial tip)

$$  Start generation of : Profile Contouring.3 (Comp. 2D Radial tip)

SPINDL/   70.0000,RPM,CLW

GOTO  /    0.00000,  155.00000,  150.00000, 0.000000, 0.000000, 1.000000

$$ START CUTCOM PLANAR XT,YT,ZT,I,J,K

      5.00000),ON,(LINE/    150.00000,    150.00000,    150.00000,$

                            155.00000,    150.00000,    150.00000)

INDIRV/    0.00000,   -1.00000,    0.00000

      5.00000),ON,(LINE/    150.00000,      0.00000,    150.00000,$

                            150.00000,     -5.00000,    150.00000)

GOTO  /    0.00000,   -5.00000,  150.00000, 0.000000, 0.000000, 1.000000

INDIRV/   -1.00000,    0.00000,    0.00000

TLON,GOFWD/      (CIRCLE/      0.00000,      0.00000,    150.00000,$

      5.00000),ON,(LINE/      0.00000,      0.00000,    150.00000,$

                             -5.00000,      0.00000,    150.00000)

GOTO  /   -5.00000,  150.00000,  150.00000, 0.000000, 0.000000, 1.000000

$$ END CUTCOM PLANAR XT,YT,ZT,I,J,K

$$  End of generation of : Profile Contouring.3 (Comp. 2D Radial tip)

1.There is no correction to calculate for CE
2.Relative user radius should be activated

 

General remarks for Cutter Compensation

•2d radial profile and 2d radial tip should not be mixed in same program
(because switch “relative user radius” is only available for complete program)
•For working with Cutter Compensation Basics use HELP(F1) of NC Infrastructure, Chapter Specify Tool Compensation
